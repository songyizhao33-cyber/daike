## 3. 管理员后台测试

### 创建管理员账号（需要手动在数据库中操作）
由于安全原因，管理员账号需要手动创建。有两种方法：

**方法A：使用MongoDB Compass（推荐）**
1. 下载并安装 MongoDB Compass: https://www.mongodb.com/try/download/compass
2. 使用你的连接字符串连接数据库：
   ```
   mongodb+srv://songyizhao33_db_user:8L0vc0KqC9zza3tf@cluster0.trauptu.mongodb.net/
   ```
3. 找到 `substitute-matching` 数据库
4. 打开 `users` 集合
5. 找到你的用户，点击编辑
6. 将 `role` 字段从 `"both"` 改为 `"admin"`
7. 保存

**方法B：使用MongoDB Shell**
```javascript
// 连接到数据库后执行
db.users.updateOne(
  { username: "你的用户名" },
  { $set: { role: "admin" } }
)
```

## 测试注意事项

1. **测试数据准备**
   - 建议创建2-3个测试账号
   - 为测试账号添加空闲时间
   - 创建一些匹配请求

2. **浏览器开发者工具**
   - 打开 F12 开发者工具
   - 查看 Console 标签检查是否有错误
   - 查看 Network 标签检查 API 请求是否正常

3. **常见问题排查**
   - 如果后端启动失败，检查 MongoDB 连接
   - 如果前端无法访问后端，检查端口是否被占用
   - 如果出现 CORS 错误，检查后端 CORS 配置
