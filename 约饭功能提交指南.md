# 约饭功能改进 - 快速提交指南

## 改进总结

本次更新改进了约饭功能的用户体验：

### 1. 智能排序 🎯
- 未过期的约饭按时间从近到远排列
- 已过期的约饭自动放在列表最后
- 已过期的约饭显示灰色文字和"已过期"标签

### 2. 删除功能 🗑️
- 新增"删除"按钮，可永久删除约饭记录
- 保留"取消"按钮，仅标记为已取消
- 删除前有二次确认，防止误操作

### 3. 时间扩展 ⏰
- 晚餐时间增加 19:00、19:30、20:00 选项

## API 变更

### 新增接口
```
PUT /api/meal/:id/cancel
- 功能：取消约饭（标记为已取消）
- 权限：需要认证，只能取消自己的约饭
```

### 修改接口
```
DELETE /api/meal/:id
- 原功能：标记为已取消
- 新功能：永久删除记录
- 权限：需要认证，只能删除自己的约饭
```

```
GET /api/meal/browse
- 原排序：按日期和时间升序
- 新排序：未过期的从近到远，已过期的在最后
```
